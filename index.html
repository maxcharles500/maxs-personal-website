Custom Professionally Built Websites

My name is Maximilian Jones, I've spent the last month learning HTML, CSS, and JavaScript. I want to utilize and push my skills to the next level by building you your next website. I will take on projects of all shapes and sizes and will be constantly striving to bring you your ideal webpage.

HTML

First I will use HTML(Hyper Text Markup Language) to create the basic markup, and layout of your website. This is where all of the content displayed on your website will go.

CSS

After I've written the basic HTML I will move onto CSS(Cascading Style Sheet). CSS is where I will style your website. This is where your input is crucial, because this is really where your website begins to look like a website. This involves anything from changing the font and size of the text, to changing the layout of how your page is displayed. For example if I had not added any CSS styling to this website it would simply be a black and white document with black text and bulleted lists.

JavaScript

Lastly I will make your website interactive and functional by manipulating the DOM(Document Object Model) using Javascript.
